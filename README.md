# Proyecto de Gestión de Librerías

Este proyecto consiste en el desarrollo de una aplicación integral para la gestión de librerías, que facilita la administración de libros, autores y préstamos. Utiliza una arquitectura en N capas para asegurar una clara separación de responsabilidades, abarcando la lógica de negocios, la capa de datos y la interfaz de usuario.

## Estructura del Proyecto

### Backend

- **Framework:** .NET ASP.NET
- **Base de Datos:** Microsoft SQL Server
- **Directorio:** `ProyectoU1`

### Frontend

- **Framework:** React
- **Directorio:** `front-web`

## Instalación

### Requisitos Previos

1. **Backend**

   - **Sistema Operativo:** Se recomienda Windows 10 u 11, macOS 10.10 o Ubuntu 16
   - **Software Necesario:**
     - .NET SDK
     - SQL Server

2. **Frontend**
   - **Node.js:** Asegúrate de tener Node.js instalado en tu sistema.

### Configuración del Backend

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/MilaProgramming/Proyecto_P1_AppsDistribuidas
   cd Proyecto_P1_AppsDistribuidas/ProyectoU1
   ```
2. Configurar la base de datos:
   Crear una base de datos en SQL Server utilizando el script proporcionado en la documentación del proyecto.

### Configuración del FrontEnd

1. Navegar al directorio del frontend
   ```bash
   cd front-web
   ```
2. Instalar dependencias
   ```bash
    npm install
   ```
3. Ejecutar la aplicación
   ```bash
    npm start
   ```

## Requisitos y Aspectos Importantes

### Requisitos Mínimos de Hardware

- **Memoria RAM:** 4 GB
- **Almacenamiento:** 10 GB de espacio disponible

### Software Recomendado

#### Backend:

- .NET SDK
- SQL Server

#### Frontend:

- Node.js
- NPM (incluido con la instalación de Node.js)

### Funcionalidades Principales

- **Gestión de Usuarios:** Añadir, editar y eliminar usuarios.
- **Gestión de Libros:** Añadir, editar y eliminar libros.
- **Gestión de Categorías:** Añadir, editar y eliminar categorías de libros.
- **Gestión de Editoriales:** Añadir, editar y eliminar editoriales.
- **Gestión de Autores:** Añadir, editar y eliminar autores.
- **Gestión de Préstamos:** Crear y gestionar préstamos de libros.
